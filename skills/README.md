# Skills Registry

This folder is for reusable, composable skills that enhance agent capabilities.

## How to add a skill

- Create a folder or `.md` file named after the skill.
- Include a `SKILL.md` file with instructions, examples, and reference material.
- Keep the skill focused on one capability.
- Reference durable knowledge files rather than duplicating project truth.
