# Scope Boundaries

## In scope (MVP)

- Authentication and role assignment
- Brother passport (four sections)
- Mentor review workflow (verify/reject/clarify)
- Mentor sessions, visitations, rituals performed logs
- Section sign-off and stage transitions
- Lodge and District dashboards
- Audit events for auth, reviews, role changes, exports, stage transitions
- Basic notifications (in-app, email/push)
- Seed data for demo lodges, roles, and Brothers

## Deferred to Release 1.1

- Offline-first sync polish
- Richer analytics and reminders
- Evidence attachments
- CSV/PDF exports
- Accessibility hardening

## Deferred to Phase 2

- Enquiry intake and contact log pipeline
- Privacy deletion workflow for enquirers

## Out of scope until authorised

- Ritual text repository
- Automatic progression decisions
- Social media / community chat
- Payment processing
- Regalia store
- Direct Solomon data integration
- Candidate acceptance automation
